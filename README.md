## Features:
- Smart Trip Planning: Input your travel details (destination, duration,
weather, and activities) through an interactive form.

- Dynamic Checklist Generation: Automatically generates a customized packing and to-do list based on your specific trip data.

- Interactive List Management: * Mark items as completed (check/uncheck).

- Remove items or entire lists.

## Current Limitations
- Edit Functionality: Currently, trip data cannot be edited once submitted. This feature is planned for future updates.
